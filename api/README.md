# API Routes

Esta pasta está preparada para futura implementação de uma API REST.

## Estrutura

```
api/
├── config/           # Configurações (Firebase, DB, etc)
├── controllers/      # Controladores da API
├── middleware/       # Middlewares (auth, validation, etc)
├── models/          # Modelos de dados
└── routes/          # Definição de rotas
```

## Arquivos Atuais

- `config/firebase-config.js` - Configuração do Firebase (não utilizado atualmente)

## Quando Implementar

Sugestões de rotas futuras:
- `/api/contact` - Envio de formulários
- `/api/analytics` - Endpoint para analytics customizado
- `/api/newsletter` - Inscrição em newsletter
- `/api/lead` - Captura de leads
