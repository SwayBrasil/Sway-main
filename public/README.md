# Frontend Assets

Esta pasta contém todos os assets do frontend (HTML, CSS, JS, imagens).

## Estrutura

```
public/
├── assets/
│   ├── css/          # Estilos CSS
│   ├── js/           # JavaScript
│   └── img/           # Imagens e favicons
└── pages/             # Páginas HTML (termos, privacidade, etc)
```

## Nota sobre GitHub Pages

Para compatibilidade com GitHub Pages, os arquivos HTML principais (`index.html`, `termos.html`, `privacidade.html`) também estão na raiz do projeto, apontando para os assets nesta pasta.

## Futuro Backend

Quando o backend for implementado, esta pasta será servida diretamente pelo servidor, mantendo a mesma estrutura de paths.
